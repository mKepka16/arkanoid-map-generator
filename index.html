Hints

  • First you need to select some
    tiles, so you can fill them with
    texture later
  • Right click on the board
    for context menu
  • Hold Shift and drag mouse
    to select block of tiles
  • Hold Ctrl and click the tile,
    to select/unselect without
    removing existing selection
  • Other shortcuts:
    Ctrl + S - Save map
    Ctrl + L - Load map
    Ctrl + Z - Undo
    Ctrl + Y - Redo
    Del - Delete tiles
